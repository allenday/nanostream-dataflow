Monitors whether PubSub is working as intended. Refreshes are needed for updates.

Common problems to look out for:

1. No messages: Either PubSub or GSC Object Change Notifications aren't working, or the app deploy environment has changed.

Test if you can send any messages from the message box (scroll all the way down).

Send message, click back, and refresh.

If the new message appears, it's Google's fault.

If no new messages appear, go back to cloud shell and re-deploy the app. This sometimes happen when you git clone to a newer version.

2. Duplicates: Doesn't seem to be anything we can do about this, it's a common bug that occurs when Google pushes new features.
